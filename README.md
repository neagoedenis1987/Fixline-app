# FixLine 🔧 — Moroni Amato

App gestione guasti macchinari sviluppata da **Neagoe Denis**.

## Come avviare

1. Installa Node.js da https://nodejs.org
2. Apri la cartella con VS Code
3. Apri il terminale e scrivi:

```
npm install
npm start
```

## Deploy su Vercel

1. Crea account su vercel.com
2. Trascina la cartella su Vercel
3. Il link è pronto!
